Type declaration

  • Optional assetsPrefix?: undefined | string

    You must specify the the base url where the components assets will be exposed.

    example

    http://localhost:3100/component-assets/

    The /component-assets/ must be the same of config.server.routes.assets

  • output: { hydrate: string; view: string }
    • hydrate: string

      Describe where hydrate compiled files should be outputted.

      default

      projectRoot/.ragu-components/output/hydrate/

    • view: string

      Describe where view compiled files should be outputted.

      default

      projectRoot/.ragu-components/output/view/

  • Optional watchMode?: undefined | false | true

    Define if ragu server should auto-recompile the project when components change.

    default

    true when ragu-server dev and false for ragu-server start.

  • webpack: { hydrate: Configuration; view: Configuration }
    • hydrate: Configuration

      The webpack configuration for hydrate (aka client-side components)

    • view: Configuration

      The webpack configuration for view (aka server-side components)

components

components: { defaultDependencies?: GlobalDependency[]; namePrefix?: undefined | string; resolver?: ComponentResolver; resolverOutput?: undefined | string; sourceRoot: string }
  • Defined in src/config.ts:48

A set of configuration for ragu components.

Type declaration

  • Optional defaultDependencies?: GlobalDependency[]

    List of dependencies which should not be incorporated at the component bundle. It is useful to avoid to have the same dependency at multiple micro-frontends.

  • Optional namePrefix?: undefined | string

    This required property aims to avoid namespace collision between micro-frontends.

  • Optional resolver?: ComponentResolver

    The resolver specifies how components will be resolved.

    default

    ByFileStructureComponentResolver

  • Optional resolverOutput?: undefined | string

    A ComponentResolver may result in auto-generated code. This property describe where the code should be generated.

    default

    projectRoot/.ragu-components/resolver-output

  • sourceRoot: string

    Describes where your ragu-components live.

    default

    projectRoot/ragu-components/
